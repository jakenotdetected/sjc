TROUBLESHOOTING:
================

If SSL 526 error persists:
  1. Check PM2 status: pm2 status
  2. Check server.js errors: pm2 logs sjc-cgu --err
  3. Verify port 3000 is listening: netstat -tlnp | grep 3000
  4. Check Cloudflare origin: https://dash.cloudflare.com

If fail2ban stopped:
  systemctl status fail2ban
  systemctl restart fail2ban
  fail2ban-client status sshd

Check security implementation:
  curl -I https://cgu.jakenetwork.xyz/api/csrf-token
  (Should return 200 with CSRF token)
