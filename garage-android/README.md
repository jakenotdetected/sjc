# Garage Android App

Garage is an Expo React Native app for Sri Lankan mechanics and customers. This first version builds the mechanic onboarding and panel.

## Run

```bash
npm install
npm run android
```

You can also run:

```bash
npm start
```

Then scan the QR code with Expo Go on Android.

## Current screens

- Mechanic / Customer role selector
- Mechanic login profile form
- Repair vehicle selector
- Service skill selector, including A/C, engine, brakes, electrical, tires, suspension, and other repairs
- Mechanic dashboard preview with profile, service tags, request cards, earnings, and quick actions

Customer mode is included as a placeholder for the next build step.
